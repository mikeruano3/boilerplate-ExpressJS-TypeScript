# PocketStoreBackend

Para iniciar la aplicación en desarrollo, ejecutar el siguiente comando:
```
npm run dev
```
Ejecutar sin demonio (NO detectar cambios en el código)
```
npm run dev:start
```
Para iniciar la aplicación en producción:
```
npm run start
```
